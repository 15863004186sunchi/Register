# Writing Skills

## Overview
Use this skill when you want to create a new skill or update an existing one.

**Announce at start:** "I'm using the writing-skills skill to create/update a skill."

## Guidelines
1. **Structure:** Use the standard Markdown format with Overview, Goals, Process, and Rules.
2. **Clarity:** Keep instructions concise and actionable.
3. **Consistency:** Follow the style and tone of existing superpowers skills.
4. **Refactor:** If a skill becomes too large, split it into sub-skills.
