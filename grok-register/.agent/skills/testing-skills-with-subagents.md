# Testing Skills with Subagents

## Overview
Use this skill when you want to verify that a new or updated skill works as expected.

**Announce at start:** "I'm using the testing-skills-with-subagents skill to verify this skill."

## Process
1. **Scenario:** Create a test scenario that exercises the skill.
2. **Dispatch:** Start a subagent with the scenario and the skill.
3. **Evaluate:** Check if the subagent followed the skill correctly.
4. **Refine:** Update the skill based on the subagent's performance.
