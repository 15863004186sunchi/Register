# Requesting Code Review

## Overview
Use this skill when you have completed a task and want to request a review of your changes.

**Announce at start:** "I'm using the requesting-code-review skill to ask for a review of my changes."

## Guidelines
1. **Prepare Diff:** Generate a clear diff of your changes.
2. **Summarize:** Write a concise summary of what you did and why.
3. **Tests:** Mention that all tests passed and provide the output.
4. **Specifics:** Ask for feedback on specific areas you're unsure about.
