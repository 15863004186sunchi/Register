# Visual Companion

## Overview
Use this skill when you need to reason about visual elements or UI changes.

**Announce at start:** "I'm using the visual-companion skill to reason about the UI."

## Guidelines
1. **Screenshots:** Take screenshots of the current state.
2. **Analysis:** Identify visual discrepancies or alignment issues.
3. **Compare:** Use the `pencil` tool to verify with the original design.
4. **Iterate:** Make changes and take a new screenshot to verify.
