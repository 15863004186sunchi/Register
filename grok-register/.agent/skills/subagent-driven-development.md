# Subagent-Driven Development

## Overview
Use this skill when you want to delegate tasks to subagents for parallel execution or specialized work.

**Announce at start:** "I'm using the subagent-driven-development skill to delegate this task to a subagent."

## Process
1. **Define Task:** Clearly state the goal, scope, and requirements.
2. **Context:** Provide the subagent with only the necessary files and information.
3. **Dispatch:** Start the subagent.
4. **Monitor:** Check progress and provide guidance if needed.
5. **Review:** Carefully review the subagent's output before merging.
