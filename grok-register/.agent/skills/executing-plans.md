# Executing Plans

## Overview
Use this skill to execute a previously written implementation plan step-by-step.

**Announce at start:** "I'm using the executing-plans skill to implement the approved plan."

## Process
1. **Load Plan:** Read the implementation plan document.
2. **Execute Step:** Perform the actions described in the current task/step.
3. **Verify:** Run the verification steps specified in the plan.
4. **Checkpoint:** Report progress and ask for feedback if needed.
5. **Next Step:** Proceed to the next step until the plan is complete.

## Rules
- Stick to the plan. If changes are needed, update the plan first.
- Verify every step. Don't assume code works.
- Commit frequently as specified in the plan.
