# Code Reviewer

## Overview
Use this skill when you want to perform a code review on a set of changes.

**Announce at start:** "I'm using the code-reviewer skill to perform a code review."

## Review Checklist
1. **Correctness:** Does the code solve the problem?
2. **Tests:** Are there sufficient tests? Do they pass?
3. **Style:** Does the code follow project conventions?
4. **Maintenance:** Is the code easy to read and maintain?
5. **Security:** Are there any obvious security vulnerabilities?
