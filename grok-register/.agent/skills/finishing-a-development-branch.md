# Finishing a Development Branch

## Overview
Use this skill when work on a feature or bug fix is complete and ready to be merged or handed off.

**Announce at start:** "I'm using the finishing-a-development-branch skill to wrap up this task."

## Checklist
1. **Verification:** Run all tests (unit, integration, end-to-end).
2. **Cleanup:** Remove temporary files, debug logs, and worktrees.
3. **Documentation:** Update READMEs, API docs, and changelogs.
4. **Review:** Perform a final self-review of all changes.
5. **Handoff:** Provide a summary of work done and any follow-up actions.
