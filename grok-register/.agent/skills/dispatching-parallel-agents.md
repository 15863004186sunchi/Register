# Dispatching Parallel Agents

## Overview
Use this skill when a task can be decomposed into independent sub-tasks that can be executed in parallel by multiple subagents.

**Announce at start:** "I'm using the dispatching-parallel-agents skill to parallelize this task."

## When to Use
- Implementing multiple independent components.
- Running long-running tasks like large-scale refactors across many files.
- Simultaneous research on different modules.

## Process
1. **Decompose:** Identify independent work streams.
2. **Prepare Context:** Create minimal context packages for each subagent.
3. **Dispatch:** Start subagents using the appropriate platform tools.
4. **Merge:** Integrate results and resolve any conflicts.
