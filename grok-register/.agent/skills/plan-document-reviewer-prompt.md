# Plan Document Reviewer Prompt

## Overview
Use this prompt to dispatch a subagent to review an implementation plan.

## Review Goals
1. **Feasibility:** Is the plan realistic and actionable?
2. **TDD:** Does the plan follow the TDD cycle (test-first)?
3. **Files:** Are all file paths and changes specified?
4. **Bite-Sized:** Are the tasks small enough for easy implementation?
